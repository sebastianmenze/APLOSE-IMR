À partir de la [liste des campagnes](all), vous pouvez accéder aux détails d'une campagne en cliquant sur sa carte.

![](/annotation-campaign/detail.png)

Vous pouvez consulter toutes les informations relatives à la campagne dans l'onglet « Information » :
- Informations générales
  - Description : Description de la campagne
  - Instruction URL : Lien vers un document externe détaillant les instructions de la campagne
  - Archive : Statut et date d'archivage (si la campagne est archivée)
  - Deadline : Date limite pour la fin de la campagne
- Informations sur les données
  - Dataset : Nom du dataset, qui est un lien vers la [page détaillée du dataset ](/user/data/manage)
  - Analysis : Tableau de l'analyse à partir du dataset utilisé dans cette campagne
- Informations sur les annotations (si des phases existent)
  - Label set : Modale de l'ensemble d'étiquettes contenant une description du dataset et la liste de ses labels.
  - Confidence set : Ensemble de niveaux de confiances et liste de ses niveaux de confiances
  - Annotation type : Types d'annotations autorisés
- Progression globale (si des phases existent)

_Certaines informations peuvent être masquées si le créateur de la campagne ne les a pas renseignées_
