Vous pouvez gérer les annotateurs d'une phase à tout moment

En haut de la page, cliquez sur le bouton Annotateurs :
![](/annotation-campaign/button-annotators.png)


Pour chaque annotateur, vous pouvez spécifier la plage de fichiers qu'il/elle peut annoter. Vous pouvez définir plusieurs plages différentes pour le même
annotateur.
Vous pouvez ajouter un nouvel annotateur à l'aide du champ de recherche.

![](/annotation-campaign/edit-annotators/form.png)

Vous pouvez également ajouter des annotateurs par groupe.

::: details Créer un groupe d'annotateurs
Accédez à la partie administration d'APLOSE à l'aide du lien « Admin ».

Recherchez « Annotator group » dans le bloc « APLOSE » et cliquez sur « Add ».

![](/campaign-creator/annotator-group/nav.png)

Vous pouvez remplir le formulaire avec le nom de votre groupe et ajouter les utilisateurs que vous souhaitez y inclure.

![](/campaign-creator/annotator-group/form.png)

Une fois votre formulaire rempli, vous pouvez l'enregistrer.

::: warning Attention

Pour l'instant, le groupe sert uniquement à ajouter plusieurs annotateurs à la fois, mais le lien entre le groupe et la campagne n'est pas conservé. Si vous ajoutez ultérieurement des annotateurs au groupe, ils ne seront pas ajoutés à la campagne !

:::

::: info

Vous pourrez ajouter des annotateurs ou des fichiers à un annotateur à tout moment pendant la campagne.

:::
